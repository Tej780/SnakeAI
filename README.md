# Deep Q Learning Snake AI

> Deep Q Learning model to play a rudimentary version of the game "Snake"

## Usage

Run in terminal:

```bash
$ make env
$ source .venv/bin/activate
$ make run
```
## Requirements
Linux (Ubuntu)
